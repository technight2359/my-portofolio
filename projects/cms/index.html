Panduan lengkap dari instalasi sampai penggunaan

1. Pendahuluan

Aplikasi Toko Online adalah platform e-commerce sederhana untuk menjual produk secara daring. Sistem ini memiliki 2 peran utama: pengguna dan admin.

2. Fitur Utama

  • Registrasi & Login Pengguna
  • Manajemen Produk
  • Keranjang & Checkout
  • Manajemen Pesanan
  • Riwayat Transaksi
  • Dashboard Admin

3. Cara Instalasi

  1. Clone repository: 
    git clone https://github.com/username/toko-online.git
  2. Masuk folder project dan install dependency: 
    cd toko-online
composer install
  3. Salin file .env dan konfigurasi database: 
    cp .env.example .env
  4. Jalankan migrasi dan seeder: 
    php artisan migrate --seed
  5. Jalankan server: 
    php artisan serve

4. Struktur Database

Tabel Deskripsi
users Data pengguna
products Data produk
orders Transaksi pesanan
order_items Detail produk per pesanan

5. Alur Penggunaan

  1. Pengguna mendaftar dan login
  2. Menelusuri produk dan menambah ke keranjang
  3. Checkout dan pilih metode pembayaran
  4. Admin mengelola dan mengirim pesanan
